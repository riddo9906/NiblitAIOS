# Niblit AIOS

This repository contains the Niblit AIOS core modules.
The NiblitCore package is a lightweight core that supports:
- module registration and dynamic loading
- event bus
- simple AI adapter stub
- configuration and basic logging

Development environment:
- Use Termux (Android) or a POSIX system.
- Use `export PYTHONPATH=~/NiblitAIOS:$PYTHONPATH` to import NiblitCore from other projects.
